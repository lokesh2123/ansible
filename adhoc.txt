We can check the ping using the command below 
-- >   ansible all -i 172.31.42.46, -e ansible_user=ec2-user -e ansible_password=DevOps321 -m ping

We can install the package on another server using the command below 
---->   ansible all -i 172.31.43.44, -e ansible_user=ec2-user -e ansible_password=DevOps321 -b -m package -a "name=nginx state=installed"

We can start the service by using below command 
------>  ansible all -i 172.31.42.46, -e ansible_user=ec2-user -e ansible_password=DevOps321 -b -m service -a "name=nginx state=started"
